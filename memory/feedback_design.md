Do not use the cosmic dark theme (dark background, glowing cyan/gold accents, starfield animations) for this site. The user explicitly rejected it as looking "too AI generated."

**Why:** User wants a minimal, elegant style like high-end Japanese websites — think Muji, museum sites, yozora39.com aesthetic translated into restraint rather than spectacle.

**How to apply:** Use white/off-white backgrounds (#ffffff, #fafaf8), thin typography (font-weight 300–400), generous whitespace, 1px borders (#e8e8e6), muted ink colors (#111111, #555555, #aaaaaa), ultra-wide letter-spacing on labels (tracking-ultra = 0.24em), no gradients, no glow effects, no rounded corners, subtle hover states (color change only, no transforms). Grid layouts with gap-px on colored backgrounds create clean dividers without border repetition.
